# Pact E2E Integration Tests

## Dependencies

```
npm
```

## Install

```
make install
```

## Create Pacts

```
make create
```

## Run Publish

Ensure you have set environment variables for `PACT_BROKER_BASIC_AUTH_USERNAME`, `PACT_BROKER_BASIC_AUTH_PASSWORD` and `PACT_BROKER_URL`

```
make publish
```
